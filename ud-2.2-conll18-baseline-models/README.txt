CoNLL 2018 Shared Task in UD Parsing
====================================

This archive contains UDPipe baseline models for CoNLL 2018 Shared Task
in UD Parsing. These models require UDPipe version 1.2 and are present
in models/ subdirectory.

The models were trained using a custom data split for treebanks where no
development data is provided. Also, we trained an additional "Mixed-UD" model,
which uses 200 sentences from every training data. All used training data can
be found in training/ subdirectory, which contains the details.

Copyright 2018 Institute of Formal and Applied Linguistics (UFAL),
Faculty of Mathematics and Physics, Charles University, Czech Republic.


Baseline model performance
--------------------------
We measure the performance using the attached official evaluation script
conll18_ud_eval.py (version 1.0).

Note that we measured the performance on the development set, which we also
used to perform hyperparameter tuning.

We report not only the official metrics, but also all other metrics,
in order for you to be able to decide whether to use UDPipe for
segmentation/tokenization/POS tagging/lemmatization. The description
of the metrics is available in the evaluation script.

We measure the performance in three settings:
- Raw text: only the raw texts are available (this is the official setting, F1 score
  is reported)
- Gold tok: gold segmentation and tokenization is available (accuracy is
  reported)
- Gold tok+mor: gold segmentation, tokenization, lemmatization and morphology
  is available (again reporting accuracy)
